Accounts and Passwords Study: Aggregated Data Samples

Test Description

  • Test name: Accounts and Passwords Study.
  • Test duration: 1 day
  • Test type: Global
  • Firefox versions covered: 3.5 and above
  • Data submission: 7823 users, between Dec 15 and Jan 7, 2009

How to read the data

Data is in CSV format. The first line of the file is the header and gives the names of all the columns. After that, each line contains a summary of the data for a single user. Here are the column names and the explanation of each one:

Column Name Explanation
happinessAnswer to survey question "How do you feel about your current options for managing accounts and passwords?". 1 is the most frustrated, 5 is the happiest.
open_idAnswer to the survey question "Do you use OpenID?" (See below for response codes)
other_toolsAnswer to survey question "Do you use other tools to help manage your password?" (See below for response codes).
multi_accounts_same_websiteAnswer to the question "Do you ever usemultiple accounts for the same wesite?" (See below for response codes)
share_computerAnswer to the question "Do you share your computer with others?" (See below for response codes)
fx_versionFirefox version string
osOperating system and version string
localeLocale string, e.g. "en-US"
num_extensionsNumber of extensions installed (not neccessarily active)
total_sitesNumber of different sites for which user has stored a password in the password manager
total_passwordsNumber of different passwords that the user has stored in the password manager
most_usedFor whichever password the user has re-used on the most sites: How many sites is that password used on?
histogramsemicolon-separated list of numbers, where each number is a number of sites that have the same password. Sorted from highest to lowest. The highest number in histogram is equal to most_used, and the number of items in histogram is equal to total_passwords.

The survey questions are mutiple-choice, so the value in those columns will be a numerical answer code. To interpret those codes, use the table below. The questions are on the left side and the answer codes are on the top. (Some questions allow multiple responses.
